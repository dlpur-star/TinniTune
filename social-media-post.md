Hey everyone,

I wanted to share a project I've been working on: **TinniTune** - a web-based tinnitus therapy application that implements evidence-based treatment protocols.

**Key Features:**

1. **Notched Sound Therapy** - Uses the clinically-validated 1.0 octave notch protocol (Okamoto et al., 2010) with proper Q-factor calculations

2. **Binaural Beats** - Multiple modes tested in clinical trials:

- Built with React and Tone.js
- Progressive Web App (works offline, installable)
